这是一个智能停车场管理系统项目的视觉算法部分

该部分功能：
1、当车辆将要进入停车场时，启动摄像头拍摄，检测识别车牌信息，上传停车场服务器
2、检测到车牌信息后启动舵机，自动控制出入口处车杆，放行车辆
3、有车辆进出变化时，更新入口处大屏上车位信息。

使用到的硬件：
1、两个摄像头（入口处和出口处）
2、单片机（控制舵机和超声波模块）
3、超声波模块
4、舵机

代码简要介绍：
1、使用的模型是weights文件夹下的plate_detect.pt，训练数据包含12类车牌，使用场景丰富
2、出入口处图片数据缓存在data文件夹下的capture_images类文件夹中
3、包含各个模块的调试代码
4、最终代码为detect_sys.py
